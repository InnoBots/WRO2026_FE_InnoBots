# System Overview

The InnoBots vehicle was developed for the WRO 2026 Future Engineers category. The objective of the project is to design an autonomous vehicle capable of navigating the competition track, detecting obstacles, and making decisions without human intervention.

Our system combines computer vision, embedded systems, and custom mechanical design. A Raspberry Pi 5 is used for image processing and navigation, while an STM32 microcontroller handles low-level control tasks such as steering and motor management. This architecture allows the robot to react quickly while maintaining stable performance during operation.

Throughout the development process, several prototypes were built and tested. Feedback from these tests helped us improve the design and create a more reliable and efficient vehicle.
